# Sorting Algorithm Visualizer

A clean React + Vite web app for visualizing how popular sorting algorithms work step by step and comparing their performance.

## Features

- Generate random arrays of vertical bars.
- Choose between five sorting algorithms.
- Animate comparisons, swaps, overwrites, and sorted values.
- Adjust array size and animation speed.
- Prevent controls from changing state while sorting is running.
- View performance metrics after each run.
- Read a short educational explanation for the selected algorithm.
- Responsive dark theme suitable for a portfolio project.

## Algorithms Included

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Shell Sort
- Counting Sort

## Technologies Used

- React
- Vite
- JavaScript
- CSS

## How to Install

```bash
npm install
```

## How to Run

```bash
npm run dev
```

Then open the local URL shown in your terminal.
